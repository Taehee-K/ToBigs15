## Problem 3: 1이 좋아

투빅이가 좋아하는 숫자는 1이다. 코로나로 심심한 투빅이는 어떤 규칙을 따라가면 특정 숫자가 1이 되는 것을 발견한다. 이를 투빅스 규칙이라고 한다. 규칙은 다음과 같다.  

    양의 정수 n을 입력 받았을 때(1<=n <= 2^68)
     1. n이 짝수라면 2로 나눈다
     2. n이 홀수라면 3을 곱하고 1을 더한다
     3. 1이 될 때까지 반복한다.
